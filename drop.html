SOME THINGS I'VE BUILT

Sweat Analyzer

The Sweat Analyzer project revolutionizes health monitoring by measuring chloride levels in sweat. Using advanced sensor technology and a user-friendly LCD display, it provides real-time and historical data for informed health decisions. Ideal for fitness enthusiasts, athletes, and those requiring continuous chloride monitoring, its portability ensures health control on the go, marking a significant advancement in personalized health technology.

Student Database

SPMS 3.0 meticulously analyzes students, course instructors, and departmental performance through a database of diverse assessments. Storing exam materials and outcomes data, it empowers students to monitor their progress and provides Higher Authorities with comprehensive analytical reports, fostering continuous improvement and informed decision-making within the educational institution.

DOHS Housing

The DOHS Housing Java project aims to streamline housing management processes using Java technologies. Leveraging object-oriented programming, it facilitates efficient data handling, user authentication, and task automation. The system prioritizes security and scalability, ensuring a robust solution for the Department of Housing and Urban Development.
